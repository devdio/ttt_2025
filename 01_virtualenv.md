# venv
- PowerShell 실행 정책 조정 (최초 1회 필요)
가상 환경의 활성화 스크립트는 기본 정책상 실행되지 않을 수 있으니, 정책을 완화해 줍니다. (관리자 권한으로 실행 권장)

```sh
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

- 가상환경 만들기
```
# 프로젝트 폴더 만들기
mkdir test_project
cd test_project

# 가상환경 만들기
python3 -m venv venv

# 가상환경 활성화
.\venv\Scripts\Activate.ps1

# 가상환경 비활성화
deactivate

```
